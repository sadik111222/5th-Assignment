# What is the difference between var, let, and const?

    In JavaScript, var, let, and const are all used to declare variables, but they behave differently in terms of scope, reassignment, and hoisting.

    1. var is the old way of declaring variables (before ES6 / 2015).

    2. let was introduced in ES6 (2015) and is the modern replacement for var.

    3. const is used for variables that should not change.

## What is the spread operator (...)?

    The spread operator (...) is a feature in JavaScript that expands (spreads) elements of an iterable (like an array, string, or object) into individual elements. It is commonly used to copy, merge, or pass values easily.

    

### What is the difference between map(), filter(), and forEach()?

    In languages like JavaScript, map(), filter(), and forEach() are array methods used to process elements in a list. The key difference is what they return and what they are used for.

    1. map()

        a. Purpose: Transform each element in an array.

        b. Returns a new array.

        c. The new array has the same length as the original.

        d. Each element is modified according to the function.

    2. filter()

        a. Purpose: Select elements that meet a condition.

        b. Returns a new array.

        c. The new array may have fewer elements.

        d. Elements are included only if the condition returns true.

    3. forEach()

        a. Purpose: Run a function for each element.

        b. Does not return a new array (returns undefined).

        c. Used for side effects like logging, updating variables, or modifying external data.

#### What is an arrow function?

    An arrow function is a shorter way to write functions in JavaScript using the => (arrow) syntax. It was introduced in ECMAScript 6 (ES6) to make code more concise and easier to read.


##### What are template literals?

    Template literals are a feature in JavaScript that make it easier to work with strings. They allow embedded expressions, multi-line strings, and string interpolation using backticks ( ) instead of single ' ' or double " " quotes.
    They were introduced in ECMAScript 2015 (ES6).
